## üÜò Common Questions & Troubleshooting

### "How do I reset my admin password?"
- Use the login page "Forgot Password" link
- Or edit directly in database if you have access
- Built-in `admin@admin.com` with password `admin` always works as backup

### "Menu item not showing on website"
- Check if **"Available"** toggle is ON in Menu Management
- Make sure image uploaded correctly
- Clear browser cache and refresh

### "Gallery image won't upload"
- Check file size (should be under 5MB)
- Use JPG or PNG format
- Check file name doesn't have special characters
- Ensure image dimensions are reasonable (not too small)

### "Coupon not working for customers"
- Check if coupon is set to **Active**
- Verify expiry date hasn't passed
- Check minimum order amount - customer's cart must exceed this
- Verify usage limit not reached

### "Customer can't place order"
- Check if menu items are marked as available
- Verify customer's address is in delivery range (if you set this)
- Test payment gateway if using online payments
- Check for any error messages in customer's browser

### "Reservations not getting confirmed"
- You must manually approve pending reservations
- Check Admin Panel ‚Üí Reservations regularly
- Set up email notifications to alert you of new bookings

### "Website looks different on mobile"
- This is normal - website is responsive
- Test on actual mobile devices
- Check that images aren't too large

### "How do I back up my data?"
- If using database, back up database regularly
- Export important data from admin panel
- Save uploaded images separately
- Contact your hosting provider for automated backups
